1. Erstelle `web/entrypoint.sh`:

```bash
#!/bin/bash
set -e

# Datenbankmigrationen ausfuehren (Pitfall 2: nie db.create_all())
echo "Running database migrations..."
flask db upgrade

# Gunicorn starten — Single Worker (Pitfall 5: APScheduler in Phase 6)
echo "Starting Gunicorn..."
exec gunicorn "app:create_app()" \
    --bind 0.0.0.0:5000 \
    --workers 1 \
    --timeout 120 \
    --access-logfile - \
    --error-logfile -
```

2. Aktualisiere `web/Dockerfile`:

```dockerfile
FROM python:3.11-slim

WORKDIR /app

# System dependencies
RUN apt-get update && apt-get install -y \
    gcc \
    && rm -rf /var/lib/apt/lists/*

# Python dependencies
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

# App files
COPY . .

# Verzeichnisse erstellen
RUN mkdir -p /app/uploads /app/data

# Entrypoint ausfuehrbar machen
RUN chmod +x /app/entrypoint.sh

# Non-root user
RUN useradd -m -u 1000 appuser && chown -R appuser:appuser /app
USER appuser

EXPOSE 5000

# Healthcheck (FOUND-05)
HEALTHCHECK --interval=30s --timeout=5s --start-period=10s --retries=3 \
    CMD python -c "import urllib.request; urllib.request.urlopen('http://localhost:5000/health')" || exit 1

ENTRYPOINT ["/app/entrypoint.sh"]
```

- `CMD ["python", "app.py"]` → `ENTRYPOINT ["/app/entrypoint.sh"]` (flask db upgrade + gunicorn)
- `mkdir -p /app/data` hinzugefuegt
- `chmod +x /app/entrypoint.sh` hinzugefuegt
- HEALTHCHECK hinzugefuegt
