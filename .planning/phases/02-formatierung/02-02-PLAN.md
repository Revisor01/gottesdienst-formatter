**Aenderung 2 — Per D-31: Multi-Termin-Zusammenfassung (Semikolon)**

Wenn ein Ort mehrere Termine am selben Tag hat, diese in EINER Zeile zusammenfassen mit '; ' als Trenner:
- Format: "Ort: 11 Uhr, Gd. m. A., Pn. Luthe; 15.30 Uhr, Kinderkirche, Pn. Luthe"

Implementierung fuer BEIDE Funktionen:
```python
# Fuer jeden Ort: Sammle alle Termin-Strings
location_entries = {}  # {location_key: [entry_strings]}

for event in day_events:
    location = ...  # wie bisher ermitteln
    time_str = format_time(event_time)
    service_type = format_service_type(event_title)
    pastor = format_pastor(event_contributor)

    entry = "{}, {}".format(time_str, service_type)
    if pastor:
        entry += ", {}".format(pastor)

    if location not in location_entries:
        location_entries[location] = []
    location_entries[location].append(entry)

# Zusammenfassen und alphabetisch sortieren
for location in sorted(location_entries.keys()):
    entries = location_entries[location]
    line = "{}: {}".format(location, '; '.join(entries))
    output_lines.append(line)
```

**Aenderung 3 — Per D-12: Zusatzinfos (anschl. Kirchcafe) anhaengen**

In der ChurchDesk-Variante: Pruefen ob der Event-Titel Zusatzinfos enthaelt.
Pattern: Wenn der Titel "anschl." oder "anschließend" enthaelt, diesen Teil nach dem Service-Typ anhaengen.
Konkretes Matching: Wenn im Originaltitel nach dem erkannten Typ-Keyword noch ein Komma mit Text folgt (z.B. "Gottesdienst mit Abendmahl, anschl. Kirchcafé"), den Teil nach dem ersten Komma als Zusatzinfo extrahieren und an den formatierten Typ anhaengen:
```python
suffix = ""
if ', anschl' in event_title.lower() or ',anschl' in event_title.lower():
    # Extract everything from ", anschl" onwards
    idx = event_title.lower().find(', anschl')
    if idx == -1:
        idx = event_title.lower().find(',anschl')
    suffix = event_title[idx:]  # includes the leading comma
```
Dann: `entry = "{}, {}{}".format(time_str, service_type, suffix)`

Fuer den Excel-Pfad analog: Das 'Titel'-Feld kann ebenfalls Zusatzinfos enthalten.
