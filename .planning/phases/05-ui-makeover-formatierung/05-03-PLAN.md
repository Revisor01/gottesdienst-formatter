   - Fehlende Felder: `<span class="text-red-500 text-xs">Fehlt</span>` (kein FontAwesome — Inline-SVG oder Text)
   - Details-Button: `<button class="text-primary-600 hover:text-primary-800 text-xs underline">`

4. **Details-Row:**
   ```html
   <tr id="details-{{ event.id }}" class="hidden">
     <td colspan="9" class="bg-gray-50 p-4">
       <div class="grid grid-cols-1 md:grid-cols-2 gap-4 text-sm">
         <!-- Beschreibung, fehlende Felder, Vollstaendigkeit, ID -->
       </div>
     </td>
   </tr>
   ```

   ```html

     <p class="text-sm mt-1">Fuer den gewaehlten Zeitraum wurden keine Gottesdienste gefunden.</p>
   </div>
   {% endif %}
   ```
