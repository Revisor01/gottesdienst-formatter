2. **web/crypto.py** — Neues Modul fuer Fernet-Verschluesselung erstellen:
   - `_get_fernet_key(secret_key: str) -> bytes`: Leitet deterministischen 32-Byte-Key aus SECRET_KEY ab via `cryptography.hazmat.primitives.kdf.pbkdf2.PBKDF2HMAC` mit SHA256, salt=b'gottesdienst-formatter' (statisch, reicht fuer diesen Anwendungsfall), iterations=480000.
   - `encrypt_value(value: str, secret_key: str) -> str`: Verschluesselt mit Fernet, gibt base64-encoded String zurueck.
   - `decrypt_value(encrypted: str, secret_key: str) -> str`: Entschluesselt, gibt Klartext zurueck.
   - Beide Funktionen nehmen secret_key als Parameter (aus `current_app.config['SECRET_KEY']` beim Aufruf).
   - Import: `from cryptography.fernet import Fernet` und `from cryptography.hazmat.primitives.kdf.pbkdf2 import PBKDF2HMAC`, `from cryptography.hazmat.primitives import hashes`, `import base64`.
